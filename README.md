# Neural-Network-project

В данном репозитории представлен код на языке Python, реализующий простейшую нейросеть для задачи классификации написанных от руки цифр датасета MNIST.

Данный код не предлагает новых методов и решений в сфере глубокого обучения, а является небольшим тренировочным проектом автора.

---

Код нейросети находится в папке `src`.

При написании кода использовались библиотеки `numpy` и `tqdm`.

В файле `neural_networks_lib.py` реализован класс нейросети `Fully_Connected_NN`, который в дальнейшем и будет использоваться для классификации.

В самом классе реализовано множество методов. Подробнее об их назначении можно прочитать в комментариях внутри кода.

Основными среди этих методов являются `train` и `test`, которые, соответственно, проводят обучение и тестирование на заданных наборах данных.

При обучении для изменения параметров модели используется стохастический градиентный спуск. Для расчета значений градиента используется алгоритм backpropogation.

В файле `supplementary_lib.py` находятся вспомогательные функции, использующиеся при написании и реализации класса нейросети (например, loss-functon).


---

Файл `read_mnist_dataset.py` является редактированным файлом с сайта kaggle: https://www.kaggle.com/code/hojjatk/read-mnist-dataset

Код данного файла реализует правильное преобразование бинарных данных MNIST в численные. Сами бинарники находятся в папке `data` и были взяты по той же ссылке с сайта kaggle.

---

В папкe `storage` в файле `NN_trained.csv` хранятся параметры одной из натренированных нейросетей. В самом классе нейросетей реализованы методы для записи и считывания таких файлов.

---

В файле `main.py` показан пример инициализации, обучения и тестирования данной нейросети.
